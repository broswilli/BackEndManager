Credits

BackendPro was created by Adam Price to provide a simple and feature full platform to start site development from. BackendPro uses a number of third party applications/libraries/code to be able to achieve all the features is provides. I just just want to give recognition the the creators of these pieces they deserve.

CodeIgniter

Without this wonderfull framework created by the great developers at EllisLab I would never have attempted to create such a large project as BackendPro. I really have a lot to thank you for, once again I now love to use PHP. CodeIgniter Link

Matchbox

Thank you Zacharias Knudsen for your wonderful module system. At first I didn't think I needed it, but it alone has kept my code tidy and easy to find. Matchbox thread Link

KhACL

Thank you Neophyte for such a wonderful library, without it I would have had to trawl through phpgacl myself. Your one library provides the bedrock of my entire application. KhACL thread Link

ReCAPTCHA + CodeIgniter Port

First of all thank you to all the people who created ReCAPTCHA and provide such a wonderful, easy to use service to protect against bots. Also thank you to JayTee for creating a CodeIgniter implementation library of the ReCAPTURE code. ReCAPTCHA Link, CodeIgniter ReCAPTCHA Implementation Link

FamFamFam Icons

Throughout BackendPro a wide range of their icons have been used to bring colour to boring operations and aid the user into what an action does. FAMFAMFAM Link

Everyone Else

Also I just want to thank the entire CodeIgniter community by first of all showing me how to love the framework and then helping out with issues and problems I had along the way
